# Zalo Automation Tool: User Guide & Documentation

## 1. Installation & Setup
The setup process is streamlined and runs automatically, adapting to your computer’s specific hardware and software configuration.

## 2. Login & Interface Overview
### Authentication
The tool utilizes a secure login method via Chromium. Upon launching the login process, the system will open a guest Chrome browser window directed to the Zalo Web URL. Users must manually log in (via QR code or phone number/password) in this window to establish the session.

### Interface Layout
* **Left Panel:** Contains core operational modules, including account management, messaging, friend requests, and group functions.
* **Right Panel:** Dedicated to application settings, system configurations, and customization options.

## 3. Messaging Functions
Designed for efficient bulk outreach and marketing campaigns, the messaging module allows you to:
* Send direct messages using a targeted list of phone numbers.
* Broadcast messages to your existing friend list.
* Send targeted messages to members within specific groups.

## 4. Friend Request Tools
Automate your network expansion with the following features:
* **Targeted Invites:** Add friends using a pre-compiled list of phone numbers.
* **Group Extraction:** Add friends directly from specific groups. The tool automatically excludes group admins to maintain a low profile and avoid bans.
* **Custom Messages:** Attach personalized welcome messages when sending friend requests to increase acceptance rates.

## 5. Group Functions
Streamline group management and community engagement:
* Join targeted groups and automate message broadcasting inside them.
* Mass invite users to your groups using targeted phone number lists.
* Automate the distribution of group invitations via links or existing friend connections.

## 6. Customization Options
Tailor the workspace to your operational preferences:
* **Language Settings:** Easily switch between Vietnamese and English interfaces.
* **Interface Themes:** Toggle between Light and Dark mode for visual comfort.
* **Layout Options:** Arrange toolbars and panels in either a Vertical or Horizontal configuration.

## 7. System Settings
Maintain stability and protect your accounts with advanced system controls:
* **IP Configuration:** Configure IP settings to ensure privacy and connection stability.
* **Proxy Support:** Full support for IP rotation via proxies or external IP-changing tools to prevent account restrictions during bulk actions.
* **Performance:** Adjust interface layout rendering and load modes for smoother operation, especially when running multiple tasks.

## 8. Final Overview
The Zalo Automation Tool is an all-in-one suite designed to maximize efficiency for marketing and outreach tasks. By centralizing account management, automated messaging, bulk friend requests, and comprehensive group tools into a highly customizable, proxy-supported interface, it provides a powerful solution for scaling your operations on the Zalo platform.
